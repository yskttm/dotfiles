---
description: メインブランチに切り替えて最新化する
---

リポジトリのメインブランチに切り替えて最新の状態に更新してください。

## 手順

1. **メインブランチ名を確認する**
   - `git symbolic-ref refs/remotes/origin/HEAD` でリモートのデフォルトブランチを取得する
   - 取得できない場合は `main` と `master` の存在を確認して判断する

2. **未コミットの変更を確認する**
   - `git status` で未コミットの変更があるか確認する
   - 変更がある場合はユーザーに提示して、以下のいずれかを選んでもらう：
     - `git stash` で一時退避する
     - 変更を破棄する（`git checkout .`）
     - 中断する

3. **メインブランチに切り替えて pull する**
   - `git checkout <メインブランチ名>` を実行する
   - `git pull` を実行する

4. **コンフリクトや失敗を確認する**
   - pull 後にコンフリクトが発生した場合はユーザーに内容を提示して対応を確認する
   - エラーが発生した場合はエラー内容を提示してユーザーに判断を仰ぐ
